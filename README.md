# 💼 Bico Certo

O **Bico Certo** é uma aplicação web desenvolvida com o objetivo de conectar trabalhadores informais de bairro a clientes que precisam de serviços.

A plataforma busca facilitar o acesso a oportunidades para profissionais como diaristas, pintores, manicures e outros prestadores de serviço, que muitas vezes têm dificuldade em encontrar clientes.

---

## 🚀 Objetivo

Criar um sistema simples e acessível que funcione como ponte entre:

- 👨‍🔧 Trabalhadores informais
- 🏠 Clientes em busca de serviços

---

## 🛠️ Tecnologias utilizadas

- HTML
- CSS
- JavaScript
- PHP
- MySQL
